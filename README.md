# Job Application Tracker

A full-stack Job Application Tracker developed using Spring Boot, Thymeleaf, MySQL, and Bootstrap.

## Features

* User Login Page
* Dashboard with Statistics
* Add Job Applications
* View Job Applications
* Update Job Applications
* Delete Job Applications
* MySQL Database Integration
* Responsive UI using Bootstrap

## Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* Thymeleaf
* MySQL
* Bootstrap
* Maven

## Screenshots

* Login Page
* Dashboard
* CRUD Operations

## Run Locally

1. Clone the repository
2. Configure MySQL in application.properties
3. Run:

mvn spring-boot:run

4. Open:

http://localhost:8080
